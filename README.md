# Day22-of-LearningCyberSecurity

# Brute force

Brute force is a trial-and-error method, often automated, that systematically tries every possible combination of characters to guess passwords, encryption keys, or login credentials to gain unauthorized access.

# Burp Suite

Burp Suite is a comprehensive platform of tools for web application security testing and penetration testing, acting as a proxy to intercept, inspect, and modify traffic between a browser and server, enabling security professionals to find vulnerabilities like SQL injection, XSS, and broken authentication.
